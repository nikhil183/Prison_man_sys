All About The Admin

The Admin plays a very important role in the Prison Mangement System.It is the only entity who have all powers in his hands regarding the PMS. The Admin will have its unique ID and password credentials.The Admin have rights to add new Jailor and to keep track on the present Jailor.He would be able to access all the entries in the database.Not only this the Admin can view all the existing prisoners which would be added by the jailor itself but can't do any kind of manipulations in the entries provided.
