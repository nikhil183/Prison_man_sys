New Management for the prisoners!!

“Prison Management System (PMS)” is an ambitious work in the same direction that will help Indian Prison System to reach those heights technologically, that it requires desperately and which will prove beneficial for the other government departments as well.This project is aimed at developing a prison management system that is a collection of registers and reports for the effective management of prisons. This system will contain the modules like nominal roll, case register, parole register, Interview requests, In-out register and an automated release diary generator.

System Login Options

Admin

Admin will see and manage all the database of the system,admin would be having access of all the records

Jailor

The jailor can record the details about the prisoner. Someone has to be there who is responsible for the security of the system.

Lawyer

Lawyer would be able to login and see the prisoner's record and other parole,case registers
